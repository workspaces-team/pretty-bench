# Examples

Specs live in `examples/specs/`.

Generated PNGs live in `examples/output/`.

Render them locally:

```bash
node ./scripts/init-examples.mjs
node ./scripts/build-renderer.mjs
node ./scripts/render-examples.mjs
```

Example formats included:

- JSON
- YAML
- TOML

Starter files generated by `node ./scripts/init-examples.mjs` are written to `examples/starters/`.
